  1.git init - initialize a new repository <br>
  2.git add filename - for saving changes  <br>
    git add . - commit changes to all files <br>
  3.git commit -m 'description' - for making changes and updating it to git  <br>
  4.git log - shows the log of recent activities <br>
  5.git checkout logaddress - for reviewing the past changes <br>
  6.git --help - offical details in wsl for reference <br>
  7.git clone : create exact copy of a repository in a new branch <br> 
  8. git mv - move the file / rename the file <br>
  9.git restore - restore working tree files <br>
  10.git rm - remove/deelete files <br>
  11.git help git - inbuilt docs within the wsl <br>
  12.git remote add origin https: - hooks the online repository locally within your device <br> 
  13.git push -u file name main - push the local changes to the online repository <br>
  14.git pull -u file name main - pull the changes made in online repository to your local device <br>
  15.git branch - shows all the branch from a repository <br>
  16.git checkout -b filename - redirects the working branch <br>
