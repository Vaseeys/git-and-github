----
  1.git init - initialize a new repository 
  2.git add filename - for saving changes 
    git add . - commit changes to all files 
  3.git commit -m 'description' - for making changes and updating it to git 
  4.git log - shows the log of recent activities 
  5.git checkout logaddress - for reviewing the past changes 
  6.git --help - offical details in wsl for reference 
  7.git clone : create exact copy of a repository in a new branch 
  8. git mv - move the file / rename the file 
  9.git restore - restore working tree files 
  10.git rm - remove/deelete files 
  11.git help git - inbuilt docs within the wsl 
  12.git remote add origin https: - hooks the online repository locally within your device 
  13.git push -u file name main - push the local changes to the online repository 
  14.git pull -u file name main - pull the changes made in online repository to your local device
  15.git branch - shows all the branch from a repository 
  16.git checkout -b filename - redirects the working branch
